# Contributing to DedeKorkutNLP

Thank you for your interest in contributing to DedeKorkutNLP! We welcome contributions from the community to help improve Turkish NLP tools.

## How to Contribute

1. **Fork the repository** to your own GitHub account.
2. **Clone the project** to your machine:
   ```bash
   git clone https://github.com/YOUR_USERNAME/DedeKorkutNLP.git
   ```
3. **Create a branch** locally with a succinct but descriptive name:
   ```bash
   git checkout -b my-new-feature
   ```
4. **Commit changes** to the branch:
   ```bash
   git commit -am 'Add some feature'
   ```
5. **Push changes** to your fork:
   ```bash
   git push origin my-new-feature
   ```
6. **Open a Pull Request** so that we can review your changes.

## Development Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run tests:
   ```bash
   python -m unittest discover tests
   ```

## Code Style

- Please follow PEP 8 guidelines.
- Add docstrings to your functions and classes.
- Write unit tests for new features.
